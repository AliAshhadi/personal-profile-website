# قالب وب‌سایت پروفایل کسب‌وکار شخصی

این یک قالب وب‌سایت پروفایل کسب‌وکار شخصی تک‌صفحه‌ای قابل استفاده مجدد است که تنها با HTML5 و CSS3 ساخته شده. این قالب دارای طراحی مینیمال و مدرن با پشتیبانی RTL برای متن فارسی، چیدمان کاملاً ریسپانسیو و برندینگ یکپارچه است. عالی برای ایجاد چندین وب‌سایت شخصی‌سازی شده با جایگزین کردن داده‌های فردی به جای placeholderها.

## ویژگی‌ها

- **چیدمان RTL**: جهت متن راست به چپ برای محتوای فارسی با کدگذاری UTF-8.
- **طراحی ریسپانسیو**: رویکرد موبایل-اول با چیدمان‌های بهینه‌سازی شده برای همه اندازه‌های صفحه.
- **بهینه‌سازی موبایل**: دکمه‌های تماس انباشته شده با متن وسط‌چین و آیکون‌های راست‌چین در صفحه‌های کوچک.
- **دسترسی‌پذیری**: HTML معنایی، برچسب‌های ARIA، ناوبری صفحه‌کلید و کنتراست رنگ کافی.
- **عملکرد**: CSS سبک، SVGهای درون‌خطی و بارگذاری سریع بدون جاوا اسکریپت.
- **تایپوگرافی**: فونت وزیر از CDN برای رندر متن فارسی سازگار.
- **عناصر تعاملی**: افکت‌های هاور و انتقال‌های نرم روی دکمه‌ها.

## ساختار

- ‏`index.html`: فایل HTML اصلی با SVGهای تعبیه شده، محتوا و متا تگ‌ها.
- ‏`style.css`: شیوه‌نامه جامع با طراحی ریسپانسیو موبایل-اول.
- ‏`architecture.md`: سند معماری و برنامه‌ریزی مفصل.
- ‏`.gitignore`: فایل‌های ناخواسته را حذف می‌کند.

## شخصی‌سازی

### تصویر پروفایل
تصویر پروفایل در حال حاضر روی `pic/account-circle-line(1).png` تنظیم شده. برای تغییر آن:
- ویژگی `src` در تگ `<img>` در `index.html` را با مسیر تصویر جدید خود جایگزین کنید (مثلاً `src="pic/your-new-image.png"`).
- اطمینان حاصل کنید که تصویر در فولدر `pic/` قرار دارد یا مسیر را به‌روزرسانی کنید.
- CSS از `object-fit: cover` برای قرار دادن تصویر در دایره بدون اعوجاج استفاده می‌کند.

### لینک‌های تماس
ویژگی‌های `href` در `index.html` را برای تلفن، ایمیل، لینکدین، واتس‌اپ و تلگرام به‌روزرسانی کنید.

### رنگ‌ها و استایل
گرادیان‌ها و رنگ‌ها را در `style.css` تغییر دهید (مثلاً تحت `.contact-btn` برای رنگ دکمه‌ها).

### محتوا
متن در `index.html` را برای نام، زیرعنوان‌ها و فوتر ویرایش کنید.

### چیدمان موبایل
نقاط شکست و اندازه‌ها را در `style.css` تحت `@media (max-width: 599px)` تنظیم کنید.

## استفاده

1. ‏`index.html` را در مرورگر مدرن (Chrome, Firefox, Safari, Edge) باز کنید.
2. ‏هیچ فرآیند ساخت یا سروری برای مشاهده محلی نیاز نیست.
3. ‏برای تولید، فایل‌ها را روی هر سرویس هاستینگ استاتیک آپلود کنید.

## الزامات

- مرورگر مدرن با پشتیبانی CSS3 (Flexbox, Grid, Transforms).
- اتصال اینترنت برای CDN فونت وزیر (اختیاری؛ به فونت‌های سیستم fallback می‌کند).
- بدون نیاز به جاوا اسکریپت، فریمورک‌ها یا ابزارهای ساخت.
